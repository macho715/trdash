각 조직이 언제 무엇을 해야 하는지 라인별로 표기. 회의 자료·책임 배분에 유용. 항목이 많아 복잡.

## 접근 방식 3가지

**A. 전행 delta shift (가장 단순)**
변경된 행의 delta(일수 차이)를 계산해, 위/아래 모든 행을 같은 delta만큼 일괄 이동.
- ✅ 구현 쉬움, 갭 유지
- ❌ 중간 작업 하나 바꾸면 전체가 덩어리째 이동

**B. Sequential chain cascade ← 추천**
변경된 행 기준으로:
- **위(backward)**: 바로 위 행부터 거슬러 올라가며 gap 유지하며 shift
- **아래(forward)**: 기존 ShiftDownstream 유지

각 행의 간격(gap)을 보존하므로, 어디서 바꿔도 전체 흐름이 자연스럽게 수축/확장됨.
- ✅ 현재 VBA에 backward pass 하나만 추가
- ✅ 행 간격(착수 준비기간) 유지
- ✅ 구현 범위 최소화 (기존 코드 최대 재활용)

**C. Predecessor DAG cascade**
COL_PRED 열에 정의된 선행 관계를 따라 위상 정렬(topological sort) 후 양방향 전파.
- ✅ 가장 정확한 의존성 반영
- ❌ 현재 workbook에 predecessor 관계가 완전히 정의되지 않음 → 별도 데이터 정비 필요
- ❌ 구현 복잡도 높음

## 설계 Section 2: 함수 상세 (`ShiftUpstream`)

공식 문서 기준으로 `ShiftDownstream`과 동일한 패턴을 역방향으로 적용합니다.

**신규 함수 (`ShiftDownstream` 바로 아래 삽입):**

```vba
Private Sub ShiftUpstream(ByVal ws As Worksheet, ByVal fromRow As Long, ByVal delta As Long)
    On Error GoTo EH

    Dim r As Long
    Dim st As Date, en As Date
    Dim useWD As Boolean: useWD = CfgOn("CFG_SHIFT_WORKDAYS_ONLY", False)

    For r = fromRow - 1 To FIRST_DATA_ROW Step -1      ' ← 역방향
        If IsGroupRow(ws, r) Then GoTo ContinueLoop

        If TryGetCellDate(ws.Cells(r, COL_START), st) And _
           TryGetCellDate(ws.Cells(r, COL_END), en) Then

            st = AddDays(st, delta, useWD)              ' DateAdd 기반 기존 helper
            en = AddDays(en, delta, useWD)

            ws.Cells(r, COL_START).Value2 = st          ' Range.Value2 (직렬 숫자)
            ws.Cells(r, COL_END).Value2   = en
            SafeSetNumberFormat ws.Cells(r, COL_START), "yyyy-mm-dd"  ' locale-neutral
            SafeSetNumberFormat ws.Cells(r, COL_END),   "yyyy-mm-dd"
            UpdateDaysFromStartEnd ws, r
        End If
ContinueLoop:
    Next r

    Exit Sub
EH:
    LogMsg "ShiftUpstream", Err.Number & " - " & Err.Description, ws.Name, fromRow
End Sub
```

**`HandleShiftMode` COL_START case 수정 (1줄 추가):**

```vba
' 기존
If delta <> 0 Then ShiftDownstream ws, r, delta, False
' 추가
If delta <> 0 Then ShiftUpstream ws, r, delta
```

**근거 (공식 문서):**
- `Range.Value2` 사용 → 날짜를 Double serial로 저장, 타입 강제변환 없이 빠름 ([docs](https://learn.microsoft.com/en-us/office/vba/api/excel.range.value2))
- `NumberFormat = "yyyy-mm-dd"` → locale 무관 포맷 ([docs](https://learn.microsoft.com/en-us/office/vba/api/excel.range.numberformat))
- `Application.EnableEvents` 가드 → 기존 `gEventProcessing` 플래그로 무한루프 방지 이미 적용됨 ([docs](https://learn.microsoft.com/en-us/office/vba/api/excel.application.enableevents))
