# 검증 체크리스트

## 메타
- 문서 목적: 로컬 실행, 퍼블릭 배포, 문서 정합성, 배포 산출물까지 포함한 표준 검증 절차를 제공합니다.
- 대상 독자: QA 담당자, 운영자, 배포 전 최종 점검자
- 기준 운영안: `kits/myagent-copilot-kit/standalone-package`
- 현재 기준 버전/산출물: `v0.1.0`, `release/myagent-copilot-standalone-v0.1.0.zip`
- 관련 문서:
  - `03-로컬-실행-가이드.md`
  - `04-퍼블릭-배포-가이드.md`
  - `../standalone-package/README.md`

## 1. 로컬 검증
- [ ] `pnpm install` 성공
- [ ] `pnpm build` 성공
- [ ] `pnpm login` 성공
- [ ] `pnpm health` 성공
- [ ] `GET /api/ai/health` 200
- [ ] `POST /api/ai/chat` 200
- [ ] 기본 모델이 `github-copilot/gpt-5-mini`로 문서와 일치
- [ ] `~/.myagent-copilot/auth-profiles.json` 생성 확인
- [ ] `~/.myagent-copilot/cache/github-copilot.token.json` 생성 또는 갱신 확인

## 2. 퍼블릭 검증
- [ ] `MYAGENT_PROXY_AUTH_TOKEN` 설정
- [ ] `MYAGENT_PROXY_CORS_ORIGINS` 설정
- [ ] 공개 엔드포인트가 HTTPS로 제공됨
- [ ] 허용 origin에서 `OPTIONS /api/ai/chat` 성공
- [ ] 비허용 origin에서 차단 확인
- [ ] 토큰 누락 시 `401` 확인
- [ ] 정상 토큰 시 `POST /api/ai/chat` 성공

## 3. DLP/라우팅 검증
- [ ] 이메일/전화번호 포함 payload에서 sanitize 또는 local route 확인
- [ ] bearer token 또는 PAT 패턴에서 block 확인
- [ ] `sensitivity=secret` 요청에서 local route 또는 안전 실패 확인
- [ ] `MYAGENT_PROXY_ALLOW_SANITIZED_TO_COPILOT=0`일 때 sanitize payload 외부 전송 금지 확인

## 4. 문서 정합성 검증
아래 사실이 모든 문서에서 동일해야 합니다.
- [ ] 기본 엔드포인트: `GET /api/ai/health`, `POST /api/ai/chat`
- [ ] 기본 모델: `github-copilot/gpt-5-mini`
- [ ] 퍼블릭 필수 env: `MYAGENT_PROXY_AUTH_TOKEN`, `MYAGENT_PROXY_CORS_ORIGINS`
- [ ] 상태 경로: `~/.myagent-copilot`
- [ ] OpenClaw fallback auth store 표현 일치
- [ ] 배포 산출물: `release/myagent-copilot-standalone-v0.1.0.zip`
- [ ] `Standalone 우선` 표현 반영
- [ ] `OpenClaw 경유는 호환 경로` 표현 반영
- [ ] `무차감 관측`과 `정책상 무제한 확정 불가` 구분 반영

## 5. 산출물 검증
- [ ] `pnpm-lock.yaml` 존재
- [ ] `dist/cli.js` 존재
- [ ] `.bat` 파일 목록이 README와 일치
- [ ] `release/myagent-copilot-standalone-v0.1.0.zip` 생성 확인
- [ ] `export-release.bat` 또는 `pnpm export:zip` 동작 확인

## 6. 샘플 명령
```bash
pnpm exec tsc -p kits/myagent-copilot-kit/standalone-package/tsconfig.json --noEmit
node kits/myagent-copilot-kit/standalone-package/dist/cli.js
node kits/myagent-copilot-kit/standalone-package/dist/cli.js health
node kits/myagent-copilot-kit/standalone-package/export-release.mjs
```
